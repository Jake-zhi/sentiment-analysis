EmotionSystemUI
	该目录下是本次系统demo的所有程序文件，Data压缩包为数据文件，MainWin.ui为系统界面，main.py为主程序。环境为python 3.6
Classifier
	该目录下是本次多分类器模型训练的源文件，classifier.py为分类器训练，Inference.py为分类器应用模块，preprocess.py为预处理源文件，PreProcessCorpus.csv为预处理好后的标签数据。环境为python 3.6
SingleModule
	该目录下是以python2.7环境为基础，完成的数据分类提取、K-Means聚类、LDA主题检测单独模块，由文件名可知程序具体功能。该目录下部分文件在EmotionSystemUI目录中已转化为python 3.6形式
数据导入与情绪分析过程.mp4文件为情绪分析的一个演示过程
